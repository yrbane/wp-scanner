Features

Why WP Scanner?

🛡️

Safe by design

Extracts WP versions and plugin metadata via regex. Never include or require — safe even on compromised servers.

⚡

Parallel scanning

Backdoor scanner uses rayon for parallel file analysis. Full site scanned in ~2 seconds.

🧠

Smart API caching

WordPress.org API responses are cached per session. Same plugin across 30 sites = 1 API call.

📄

4 output formats

console (colored), json, md, html — dark-themed standalone HTML page included.

📧

Email reports

Auto-detects local postfix/sendmail. Falls back to remote SMTP. Just add --mail.

🔄

Updatable patterns

Backdoor patterns stored in an external JSON file. Update with wp-scanner update — no rebuild needed.

Commands

What it does

📋
list
wp-scanner list [--site filter]
Discover all WordPress installations served by nginx.
🔢
versions
wp-scanner versions [--site filter]
Check WordPress core versions against the latest release. Flags outdated installs.
🧩
plugins
wp-scanner plugins [--site filter]
Scan installed plugins, check for updates via wordpress.org API. Suggests wp plugin update commands.
🔐
permissions
wp-scanner permissions [--site filter]
Audit file ownership and permissions. Detects world-writable dirs, www-data ownership. Generates fix commands.
🔍
backdoor
wp-scanner backdoor [--site filter]
Scan for PHP backdoors, webshells, obfuscated code, and suspicious files. 38 patterns, parallelized with rayon.
📊
report
wp-scanner report --format html --mail admin@example.com
Comprehensive security report combining all checks. Supports console, JSON, Markdown, and HTML output.
🔄
update
wp-scanner update
Download the latest backdoor patterns from wp-scanner.gie.im. No rebuild needed.

Backdoor detection

Pattern database

Patterns are stored in an external patterns.json file and can be updated independently from the binary.

Update patterns at any time:
$ wp-scanner update
Downloads the latest patterns from wp-scanner.gie.im/patterns.json to ~/.config/wp-scanner/patterns.json

Getting started

Installation

1

Install Rust

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2

Build

git clone https://github.com/yrbane/wp-scanner.git
cd wp-scanner && cargo build --release

3

Install

cp target/release/wp-scanner ~/.local/bin/

install 
$ git clone https://github.com/yrbane/wp-scanner.git
$ cd wp-scanner
$ cargo build --release
   Compiling wp-scanner v0.1.0
    Finished release [optimized] target(s) in 15.2s
$ cp target/release/wp-scanner ~/.local/bin/
$ wp-scanner --version
wp-scanner 0.1.0
